Pasos de la creación de proyecto
•	Se crea un entorno virtual llamado venv 
•	Activamos el entorno virtual venv 
•	Luego instalamos django con pip
•	Se crea un proyecto con su respectivo nombre 
•	Se crea la app llamada coder
•	Se registra la app coder en settings.py 
•	Se crea urls.py en dentro de la app y ponemos la url que redirigirá a la view
•	En views.py creamos una función para renderizar el html
•	Creamos templates y adentro una carpeta llamada coder y tendrá el archivo coder.html
•	Configuramos las urls en el proyecto importando el include y crear el link que nos dirigirá a la plantilla renderizada de html en este caso coder.html
